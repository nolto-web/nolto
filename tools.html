NOLTO | ツール紹介

AIとWeb制作に使っている
おすすめツール

NOLTOの制作過程で使用・検討したツールを、カテゴリ別にまとめて紹介しています。
用途とおすすめする人を合わせて記載しているので、選ぶ際の参考にどうぞ。

このページでは、NOLTOの制作過程で使用・検討したツールを紹介しています。一部リンクにはアフィリエイト広告を含む場合があります。掲載内容は個人の制作環境や目的に基づくものであり、すべての方に同じ結果を保証するものではありません。

AIコーディングツール

HTMLやCSS、JavaScriptのコーディングをAIに補助してもらうためのツールです。LP制作のコーディング効率化に活用しています。

💬

Claude Code

Anthropic

AIを使ってHTMLやCSS、JavaScriptを生成・編集できるCLIツール。ファイルの作成・読み込み・修正も自動化でき、LP制作の効率が大幅に上がる。NOLTOのメイン開発環境。

用途 LP・フォーム・サイトのコーディング全般
おすすめ AIを使いながらサイト制作を進めたい人

OpenAI Codex

OpenAI

OpenAIが提供するコード生成AI。テキストで指示するだけでコードを自動生成・補完でき、短いスクリプト作成やアイデア出しに使いやすい。

用途 コード補完・短いスクリプトの生成
おすすめ 短いコードをすぐに作りたい人
公式サイトを見る →

Cursor

Cursor AI

AIが組み込まれたコードエディタ。コードを書きながらAIに質問・修正・生成を依頼できる。VS Codeベースなので操作感が馴染みやすく、初めてのエディタにも向いている。

用途 AIを使いながらのコーディング・修正
