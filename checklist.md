## 4. SEO・メタ情報

- [ ] `<title>` タグに実際のサービス名・会社名が入っているか
- [ ] `<meta name="description">` の説明文が適切か（120文字前後）
- [ ] OGP（SNSシェア用）タグが必要な場合は追加した
  ```html
  <meta property="og:title" content="AI集客スターターパック">
  <meta property="og:description" content="中小企業の集客導線を低コストで整えます。">
  <meta property="og:image" content="https://your-domain.com/ogp.png">
  <meta property="og:url" content="https://your-domain.com/">
  ```
- [ ] `favicon.ico` を追加した（未設定だとブラウザタブにデフォルトアイコンが表示される）

---

## 6. GitHub Pages 公開手順

```
1. GitHubで新しいリポジトリを作成（例: ai-sales-lp）
2. 以下のファイルをリポジトリのルートにアップロード
      index.html
      style.css
      script.js
3. Settings → Pages → Source を「main branch」に設定
4. 数分後に https://<ユーザー名>.github.io/<リポジトリ名>/ で公開される
```
