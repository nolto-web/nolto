## 7. GitHub Pages 公開手順

```
1. GitHubで新しいリポジトリを作成（例: koumuten-lp）
2. 以下のファイルをリポジトリのルートにアップロード
      index.html
      style.css
      script.js
3. Settings → Pages → Source を「main branch / root」に設定
4. 数分後に https://<ユーザー名>.github.io/<リポジトリ名>/ で公開される
```
